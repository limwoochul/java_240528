
-git의 작업 순서
1. 저장소 생성
2. 저장소를 클론을 통해 복제 (클론 = 저장소->내 PC로 가져오는 것)
3. 작업(코딩)
4. 커밋(원격 저장소에 업로드할 내용을 정리) → 현재 작업하는 내 pc에서만 저장
	원격 저장소에는 변함이 없음.
5. 푸쉬(정리한 내용을 원격 저장소에 업로드)

*숨김 폴더 .git 폴더를 삭제하는 경우 클론이 불가능(이 경우 다시 클론을 통해 복제)


-git 이용해 집에서 복습하기.
1. git 폴더(학원)에 클론
2. git 폴더에 작업
3. git 폴더를 커밋 후 푸쉬

4. home git(집)에 클론
5. home git에 복습
6. home git을 커밋 후 푸쉬

	(home git → git 저장소 변경)
7. git 폴더에 fetch를 클릭
8. git 폴더에 pull을 클릭(원격 저장소에 있는 내용을 내 저장소로 내려받기)
9. git 폴더에 작업
10. git 폴더에 커밋 후 푸쉬

11. home git에 fetch 후 pull
12. home git에 복습
13. home git에 커밋 후 푸쉬
